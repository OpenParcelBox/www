# OpenParcelBox Website

GitHub Pages website for OpenParcelBox.

Canonical domain:

https://openparcelbox.org

Fallback GitHub Pages URL:

https://openparcelbox.github.io/www/

## DNS

The domain is managed at Cloudflare. Point `openparcelbox.org` to GitHub Pages.

Recommended records:

```text
A     @     185.199.108.153
A     @     185.199.109.153
A     @     185.199.110.153
A     @     185.199.111.153
CNAME www   openparcelbox.github.io
```

In GitHub Pages, the custom domain should be:

```text
openparcelbox.org
```

## Assets

- `assets/openparcelbox-icon.svg`: icon-only mark for favicon/avatar use
- `assets/openparcelbox-logo-horizontal.svg`: horizontal logo
- `assets/openparcelbox-social.svg`: social preview / hero preview
